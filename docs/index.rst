loanpy
======

**loanpy** is a pure-Python toolkit for loanword-oriented historical linguistics:
segment clustering, etymological alignment, sound-correspondence mining, and
adaptation (substitution plus phonotactic repair). Version 4 has no third-party
runtime dependencies.
